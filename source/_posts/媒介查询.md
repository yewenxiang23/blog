### 使用CSS3的 Media Query 查询

1.外链方式

```
<link type="text/css" rel="stylesheet" href="link.css" media="only screen and (max-width:480px)"/>
```
只有当屏幕宽度 <= 480px 时，才会加载这个CSS文件。

2.内嵌方式

```css
@media screen and (min-width: 480px){
	body{
		background:blue;
    }
}
```
当屏幕宽度  >= 480 时 body 背景为蓝色。
